SAFE TASK — Make Hire Me Review Page Read-Only and Route Counters to Existing 3-Tab Flow

Goal:
The Hire Me “Review This Agent’s Proposed Terms” page should be a clean review/decision page, not an editable negotiation form.

Current issue:
Services are editable with checkboxes on the review page, while Broker Compensation & Agency Agreement Terms are read-only. This creates inconsistent UX.

Correct behavior:
1. Review page:
   - Services display as grouped bullet points only
   - No service checkboxes
   - Broker Compensation & Agency Agreement Terms remain read-only
   - User can either accept, request changes/counter, or cancel

2. Counter flow:
   - If user clicks “Request Changes / Counter Terms,” send them to the existing Hire Agent counter flow with the 3 tabs:
     - Offered Services
     - Additional Details
     - Broker Compensation & Agency Agreement Terms
   - In that counter flow, services should be editable, and compensation/agreement terms should be editable.

- Existing counter route only if needed to connect button
