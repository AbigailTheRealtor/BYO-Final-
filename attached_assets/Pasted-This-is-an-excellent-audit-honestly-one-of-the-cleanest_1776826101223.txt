## Objective

Create a **read-only Hire Me widget** that can be embedded on external websites via iframe.

This widget must:

* show a simple agent card
* use preset data
* link into the existing Hire Me flow
* not perform any writes or require authentication

---

## Step 1 — Create Widget Controller

Create:

```id="widget-controller"
app/Http/Controllers/WidgetController.php
```

Add method:

```php
public function show($agentShortId, $role, $propertyType = null)
```

### Responsibilities:

* resolve agent using `short_id`
* load AgentDefaultProfile for:

  * role
  * property type
* validate preset:

  * must exist
  * must have services > 0

### If invalid:

* return 404 OR simple “not available” view

### If valid:

* pass data to widget view:

  * agent name
  * brokerage (if exists)
  * role label
  * property type label
  * service count
  * clean public Hire Me URL

---

## Step 2 — Add Public Widget Route

In `routes/web.php`:

Add **outside auth middleware**:

```php
Route::get('/widget/hire/{agentShortId}/{role}/{propertyType?}', [WidgetController::class, 'show'])
    ->where('agentShortId', '[0-9a-f]+')
    ->name('hire.agent.widget');
```
