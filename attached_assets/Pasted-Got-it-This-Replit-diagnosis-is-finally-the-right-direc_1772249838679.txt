You’re hitting a very specific Livewire + jQuery trap:

* **jQuery** `trigger('input')` does **not** fire the **native** input event the way Livewire expects in many setups.
* So your hidden JSON inputs **never notify Livewire**, meaning:

  * `property_items_json` never updates → `property_items` never updates → your Blade `@if(in_array('Other', $property_items))` never becomes true → the “Other” text box never shows.
  * Same for `condition_prop_buyer_json` and `number_of_unit_type_json` → nothing saves.

### The fix

You must dispatch **native DOM events** on the hidden inputs (InputEvent/HTMLEvent), not jQuery events.

Below is the **one copy/paste block** you can put into Replit. It’s written so you can paste it once and reuse it in both Buyer create + Buyer edit scripts.

---

## COPY/PASTE INSTRUCTIONS — FIX JSON BRIDGE EVENT DISPATCH (Livewire MUST receive native events)

### 1) Update `setJsonModel()` in BOTH buyer scripts

Find `setJsonModel()` wherever you added it (Buyer Create + Buyer Edit). Replace it completely with this:

```js
function setJsonModel(modelName, arr) {
  const json = JSON.stringify(arr || []);

  // Prefer hidden input bridge if present
  const input = document.querySelector(`input[wire\\:model\\.defer="${modelName}"]`);

  if (input) {
    input.value = json;

    // Fire native events Livewire listens for
    input.dispatchEvent(new Event('input', { bubbles: true }));
    input.dispatchEvent(new Event('change', { bubbles: true }));

    return;
  }

  // Fallback: direct Livewire set
  if (typeof @this !== 'undefined') {
    @this.set(modelName, json);
  }
}
```

Important:

* Do **not** use `$(input).trigger('input')`
* Do **not** use jQuery `.trigger()` at all for the hidden bridge

### 3) Ensure your Select2 change handlers call setJsonModel()

Still in your buyer scripts (create + edit), make sure these are present:

```js
// Conditions
if ($('#condition_prop_buyer').length) {
  if (!$('#condition_prop_buyer').hasClass('select2-hidden-accessible')) {
    $('#condition_prop_buyer').select2({ width: '100%' });
  }

  $('#condition_prop_buyer')
    .off('change.jsonCond')
    .on('change.jsonCond', function () {
      setJsonModel('condition_prop_buyer_json', $('#condition_prop_buyer').val() || []);
    });
}

// Income unit type
$('.number_of_unit_type').each(function () {
  const $el = $(this);

  if (!$el.hasClass('select2-hidden-accessible')) {
    $el.attr('multiple', 'multiple');
    $el.select2({ width: '100%' });
  }

  $el.off('change.jsonUnits').on('change.jsonUnits', function () {
    setJsonModel('number_of_unit_type_json', $el.val() || []);
  });
});

// Property items
if ($('#property_items').length) {
  if (!$('#property_items').hasClass('select2-hidden-accessible')) {
    $('#property_items').select2({ width: '100%' });
  }

  $('#property_items')
    .off('change.jsonItems')
    .on('change.jsonItems', function () {
      setJsonModel('property_items_json', $('#property_items').val() || []);
    });
}
```
