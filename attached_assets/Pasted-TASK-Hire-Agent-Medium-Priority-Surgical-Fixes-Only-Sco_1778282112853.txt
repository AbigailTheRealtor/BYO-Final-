Fix 1 — FAIL-04: Buyer and Landlord service tabs use Tenant wording

Files to inspect and likely fix:
- resources/views/livewire/hire-buyer-agent/hire-buyer-agent.blade.php
- resources/views/livewire/hire-buyer-agent/hire-buyer-agent-edit.blade.php
- resources/views/livewire/hire-landlord-agent/hire-landlord-agent.blade.php
- resources/views/livewire/hire-landlord-agent/hire-landlord-agent-edit.blade.php

Problem:
Buyer and Landlord service sections contain visible service text written from a Tenant perspective, such as:
- “List the Tenant's rental criteria on BidYourOffer.com.”
- “Market the Tenant's rental criteria…”
- “Promote the Tenant's rental criteria…”

Expected:
Buyer flow should use Buyer/purchase-facing language.
Landlord flow should use Landlord/property/listing-facing language.

Buyer wording rules:
- Replace “Tenant” with “Buyer” only when the service is clearly about the Buyer’s purchase criteria.
- Use “purchase criteria” instead of “rental criteria.”
- Use “properties for sale” / “purchase goals” / “Buyer’s target property criteria” where appropriate.
- Do NOT use lease/rental wording unless the specific field is intentionally lease-option related.

Examples:
Wrong:
“List the Tenant's rental criteria on BidYourOffer.com.”
Correct:
“List the Buyer’s purchase criteria on BidYourOffer.com.”

Wrong:
“Market the Tenant's rental criteria…”
Correct:
“Market the Buyer’s purchase criteria…”

Wrong:
“Promote the Tenant's rental criteria…”
Correct:
“Promote the Buyer’s purchase criteria…”

Landlord wording rules:
- Replace Tenant-facing “rental criteria” language with Landlord/property/listing-facing language.
- Use “available rental property,” “Landlord’s listing,” “lease listing,” or “rental offering” where appropriate.
- Do NOT make it sound like the Landlord is looking for a rental.
- Do NOT use Buyer/Seller purchase wording unless the section is intentionally lease-option/sale related.

Examples:
Wrong:
“List the Tenant's rental criteria on BidYourOffer.com.”
Correct:
“List the Landlord’s available rental property on BidYourOffer.com.”

Wrong:
“Market the Tenant's rental criteria…”
Correct:
“Market the Landlord’s rental listing…”

Wrong:
“Promote the Tenant's rental criteria…”
Correct:
“Promote the Landlord’s available property…”

Important:
- Make the same wording corrections in both create and edit blades.
- Keep service structure, checkbox names, array keys, and wire:model bindings unchanged unless absolutely necessary.
- Do not remove services.
- Do not add new services.
- Only correct visible labels/descriptions/tooltips that are clearly wrong-role Tenant wording.
