✅ Step 2 – Treat Income the same as Residential for Buyers

At the top of the buyer section, add this once:

@php
    $userType  = $user_type ?? null;
    $service   = $service_type ?? null;
    $type      = strtolower($property_type ?? '');
    $isBuyer   = $userType === 'buyer';
    $isFull    = $service === 'full_service';

    // For now, Income should behave exactly like Residential for Buyers
    $isBuyerResidentialOrIncome = in_array($type, [
        'residential',
        'residential property',
        'income',
        'income property',
    ]);
@endphp


Then, for each buyer tab pane (Purchasing Terms, Services, Additional Details,
Broker Compensation, Buyer Information), wrap the buyer content like this:

@if ($isBuyer && $isFull && $isBuyerResidentialOrIncome)
    {{-- existing BUYER tab content goes here --}}
    {{-- this can be @include(...) calls or inline markup --}}
@endif


Important:
Do not leave any old conditions like:

@if ($property_type === 'Residential')
    ...
@endif


around the buyer tab content. Those are what make Income show up blank.
Replace them with the $isBuyerResidentialOrIncome check above.

Keep the existing buyer @include calls.
