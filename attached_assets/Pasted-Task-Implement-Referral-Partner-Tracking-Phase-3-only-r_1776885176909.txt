PHASE 3 — REFERRAL LINK SERVICE ONLY

1. Create:
- app/Services/ReferralLinkService.php

2. Add these methods:

A. getOrCreateForAgent(int $agentId): array
Behavior:
- Look in agent_referral_links for an existing active row for this agent.
- If found, return it.
- If not found, generate a unique code, insert a new active row, then return it.
- Return enough data for later use, such as:
  - id
  - agent_id
  - code
  - is_active
  - url

B. generateUniqueCode(int $agentId): string
Behavior:
- Create a short, human-shareable code.
- Prefer a readable format, for example:
  - AGT{agentId}{random}
  or
  - short alpha prefix + short random suffix
- Must be unique against agent_referral_links.code
- Retry safely if collision occurs
- Keep it reasonably short and shareable

C. buildUrl(string $code): string
Behavior:
- Build the full absolute URL using the existing referral.capture route
- Use route('referral.capture', ['code' => $code]) if appropriate in this project
- Output should resolve to:
  /invite/{code}
