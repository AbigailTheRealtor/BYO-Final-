# Step 3 — Add a single reusable “Selected Services Only” component

Create this file:

**`resources/views/components/selected-services-only.blade.php`**

Paste this EXACT component:

```blade
@props([
    'selected' => null,
    'emptyText' => 'None Selected',
])

@php
    // Accept array or JSON string; DO NOT modify the stored value.
    $raw = $selected;

    if (is_string($raw)) {
        $decoded = json_decode($raw, true);
        if (json_last_error() === JSON_ERROR_NONE) {
            $raw = $decoded;
        }
    }

    // Normalize into either:
    // A) categorized array: [ "Category" => ["Service A","Service B"] ]
    // B) flat list: ["Service A","Service B"]
    $isArray = is_array($raw);

    $flat = [];
    $categorized = [];

    if ($isArray) {
        $isFlatList = count($raw) === count(array_filter($raw, 'is_string'));

        if ($isFlatList) {
            $flat = array_values(array_unique(array_filter(array_map('trim', $raw))));
        } else {
            foreach ($raw as $category => $items) {
                if (!is_array($items)) continue;
                $itemsClean = array_values(array_unique(array_filter(array_map('trim', $items))));
                if (count($itemsClean)) {
                    $categorized[$category] = $itemsClean;
                    foreach ($itemsClean as $s) $flat[] = $s;
                }
            }
            $flat = array_values(array_unique(array_filter($flat)));
        }
    }

    $hasCategorized = count($categorized) > 0;
    $hasAny = $hasCategorized || count($flat) > 0;
@endphp

@if($hasAny)
    @if($hasCategorized)
        @foreach($categorized as $category => $items)
            <div class="mt-4">
                <div class="font-semibold">{{ $category }}</div>
                <div class="mt-2 flex flex-wrap gap-2">
                    @foreach($items as $service)
                        <span class="px-3 py-1 rounded-full border text-sm">{{ $service }}</span>
                    @endforeach
                </div>
            </div>
        @endforeach
    @else
        <div class="mt-2 flex flex-wrap gap-2">
            @foreach($flat as $service)
                <span class="px-3 py-1 rounded-full border text-sm">{{ $service }}</span>
            @endforeach
        </div>
    @endif
@else
    <div class="text-sm opacity-70">{{ $emptyText }}</div>
@endif
```

✅ This component safely handles:

* JSON string selections
* flat arrays
* categorized arrays
* empty values

…and it **never** touches how things are saved.
