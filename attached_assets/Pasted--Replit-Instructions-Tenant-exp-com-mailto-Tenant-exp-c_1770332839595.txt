## A2) Ensure the Edit wizard uses the SAME “single increment” logic as Create

### Requirement

Edit wizard must have:

* ONE state variable controlling current tab (`$activeTab` or `$currentStep`)
* ONE Livewire method for Next (`goNext()`), one for Back (`goBack()`)

### Fix (Edit Livewire)

Add to **TenantAgentAuctionEdit.php** (or match your naming):

```php
public int $activeTab = 1;
public bool $isNavigating = false;

public function goNext(): void
{
    if ($this->isNavigating) return;
    $this->isNavigating = true;

    // Optional: validate only current tab here
    // $this->validateCurrentTab();

    $this->activeTab = min($this->activeTab + 1, $this->maxTab());

    $this->dispatchBrowserEvent('tenant-edit-nav-unlock');
}

public function goBack(): void
{
    if ($this->isNavigating) return;
    $this->isNavigating = true;

    $this->activeTab = max($this->activeTab - 1, 1);

    $this->dispatchBrowserEvent('tenant-edit-nav-unlock');
}

public function maxTab(): int
{
    return 8; // set to actual number of edit tabs
}
```

### Unlock handler (Edit Blade only)

In `tenant-agent-auction-edit.blade.php` add:

```html
<script>
  window.addEventListener('tenant-edit-nav-unlock', () => {
    setTimeout(() => {
      @this.set('isNavigating', false);
    }, 50);
  });
</script>
```

✅ This prevents double-advances even if duplicate click handlers exist.

## B3) Livewire: implement deletePersonalPhoto safely

### Location

`TenantAgentAuctionEdit.php`

```php
public function deletePersonalPhoto(): void
{
    $auction = $this->getAuctionModel(); // however you load it in edit component
    $path = $auction->personal_photo_path ?? null;

    if (!$path) {
        $this->existingPersonalPhotoPath = null;
        return;
    }

    // SAFEGUARD: only delete if file exists
    if (\Illuminate\Support\Facades\Storage::exists($path)) {
        \Illuminate\Support\Facades\Storage::delete($path);
    }

    // Clear DB + component state
    $auction->personal_photo_path = null;
    $auction->save();

    $this->existingPersonalPhotoPath = null;
    $this->personal_photo = null;

    $this->dispatchBrowserEvent('notify', ['type' => 'success', 'message' => 'Photo deleted.']);
}
```

✅ This deletes only the listing’s file and clears DB reference.
