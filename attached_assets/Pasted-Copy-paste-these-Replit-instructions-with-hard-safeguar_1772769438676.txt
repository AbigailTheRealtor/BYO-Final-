This means the visible UI selection is **not staying synced** to the Livewire property.

# LIKELY ROOT CAUSE

This is most likely one or more of these:

1. The Select2 field is being re-rendered by Livewire and losing state
2. The field is `wire:model` bound directly while also being managed by Select2
3. The Livewire property type does not match the field type
4. The JS change handler is not pushing the value back into Livewire
5. Validation is checking a different property name than the one Select2 updates
