AUDIT FIRST:
For every Select2 field in Seller, Buyer, Landlord, Tenant, report:
1. Role
2. Tab
3. Field label
4. Blade file
5. Select id/class
6. Single-select or multi-select
7. Current data-placeholder value
8. Current JS initialization function
9. Whether it currently displays “Select”
10. Whether it has an empty first <option></option> if single-select

FIX RULE:
Every empty Select2 field must visibly show “Select.”

For ALL Select2 fields:

1. Blade must include:
data-placeholder="Select"

2. JS initialization must include:
placeholder: $(element).data('placeholder') || 'Select',
width: '100%',
allowClear: true

3. Single-select Select2 fields must include an empty first option:
<option></option>

4. Multi-select Select2 fields do not need an empty option, but must have:
data-placeholder="Select"
and placeholder config in JS.

5. Guard against double init:
if (!$(element).hasClass('select2-hidden-accessible')) {
    // initialize Select2
}

IMPORTANT:
Do not only add width. Width fixes icon/collapse issues, but placeholder text also requires placeholder config and, for single-select Select2, an empty first option.
