* `Garage Needed` = Yes

```

Number of Garage Spaces Needed
```

do **not appear**, even though values exist.

This is caused by the **conditional rendering logic not recognizing stored values in edit mode**.

---

The numeric inputs must appear when:

```

OR

```

```
OR

```

* saved values display correctly
