### On first load of an agent bid form:

Populate fields in this order:

1. **Existing saved bid draft / existing bid record**

2. **Listing-derived values**
   For compensation/agreement terms that come from the listing creator, use listing values where applicable.

3. **Agent profile defaults**

### Never override existing saved bid data

* do not replace their data with profile defaults
* do not replace their edits with listing defaults
* autopopulate should be first-load only for empty fields
