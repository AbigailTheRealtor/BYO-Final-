FIX AND STANDARDIZE LISTING-PAGE BID CARDS (ALL 4 ROLES)

Goal:
Make all 4 listing-page bid cards match the same complete structure, using the Tenant/Buyer Match Score box as the depth standard and applying it consistently across:
- Hire a Tenant's Agent
- Hire a Landlord's Agent
- Hire a Buyer's Agent
- Hire a Seller's Agent

This is a bid-card rendering/standardization task.
Do not redesign the platform.
Do not create separate card styles by role.

==================================================
FINAL UX RULE
==================================================

Do NOT add a separate "Offered Terms" summary line under Offered Services.

Keep the card structure as:

1. Offered Services summary line
2. Extra Value Added line
3. Detailed Match Score box
4. Broker Compensation Summary
5. View Full Bid
6. Edit Bid when allowed

Reason:
- Services are best summarized in a single scan line
- Terms belong inside the Match Score box
- A separate Offered Terms line would be redundant and clutter the card

==================================================
REQUIRED UNIVERSAL BID CARD STRUCTURE
==================================================

All 4 roles must render the same card structure in the same order:

1. Header row
- Agent label/name
- status badge

2. Offered Services summary line
Format:
Offered Services: X/Y matched • X extra • X missing

3. Extra Value Added line
Format:
Extra Value Added: X Services
— does not affect match score

4. Detailed Match Score box
Must match the more complete Tenant/Buyer style, not the simplified Seller/Landlord version.

5. Broker Compensation Summary
Use this exact heading for all roles:
Broker Compensation Summary:

Do NOT use:
- Buyer's Broker Compensation Summary
- Seller's Broker Compensation Summary
- Landlord's Broker Compensation Summary
- Tenant's Broker Compensation Summary

The heading should be universal:
Broker Compensation Summary:

The role-specific compensation labels/details can remain inside the section.

6. View Full Bid

7. Edit Bid
- only when appropriate for the bidding agent
- keep current permission rules

==================================================
MATCH SCORE BOX — REQUIRED STANDARD
==================================================

Seller and Landlord Match Summary currently are too simplified.
They must be upgraded to match the more detailed Tenant/Buyer Match Score box format.

Use one detailed format for all 4 roles:

- Match Score heading

- Services Match: X%
- Matched: X/Y

Use the more detailed format already visible in Tenant/Buyer as the structural standard.
