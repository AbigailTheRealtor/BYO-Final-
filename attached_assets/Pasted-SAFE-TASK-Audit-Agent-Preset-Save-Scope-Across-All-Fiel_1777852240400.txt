SAFE TASK — Audit Agent Preset Save Scope Across All Fields, Roles, and Property Types

Issue:
On Edit Preset: Hire a Buyer Agent, selecting “All Buyer presets” did not save Broker Compensation & Agency Agreement Terms across all Buyer property types.

Expected behavior:
The save-scope dropdown should control how the preset form saves.

If user selects:

1. This preset only:
- Save every field on the form only to the current role/property type preset.

2. All [Current Role] presets:
- Save every field on the form to all presets for the same role.
- Example: All Buyer presets should save all form fields across:
  - Buyer Residential
  - Buyer Income
  - Buyer Commercial
  - Buyer Business
  - Buyer Vacant Land

3. All roles and property types:
- Save fields that are truly shared/global safely across all role/property presets.
- Audit first before changing this behavior because services and compensation may not map cleanly across roles.

IMPORTANT:
Do not assume only public profile fields should propagate anymore.
Audit what each button actually saves and compare it to expected behavior.
