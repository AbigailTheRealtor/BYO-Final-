# REPLIT INSTRUCTIONS WITH SAFEGUARDS — HIRE A BUYER’S AGENT — OFFERED FINANCING/CURRENCY SHOULD ONLY SHOW CURRENTLY SELECTED OPTIONS

## GOAL

Fix the **Hire a Buyer’s Agent** listing display so that under **Offered Financing/Currency**, the listing only shows the financing options that are **currently selected and saved**.

## CURRENT ISSUE

When editing a Buyer Agent listing, if the user removes some financing options, the listing still continues to show old removed options.

Example:

* user originally had **Conventional, FHA, Jumbo**
* user clicks **Edit Listing**
* user removes **FHA** and **Jumbo**
* user saves edits
* listing should show only **Conventional**
* right now it still shows removed options, which is wrong

## REQUIRED RESULT

For these financing options:

* Conventional
* FHA
* Jumbo
* VA
* No-Doc
* Non-QM
* USDA

the actual listing should display **only** the values that are currently selected after edit/save.

If an option is unselected and saved, it must **not** continue showing on the listing.
