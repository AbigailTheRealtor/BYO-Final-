# STEP 2 — Wrap it in a Residential check

We only want this to show if property is **commercial**.

### Replace the existing line:

```blade
<div>
  <strong>Owner Responsible For:</strong>
  {{ $listing->owner_responsible_for }}
</div>
```

### WITH THIS:

```blade
@php
  $isCommercial = false;

  // Adjust to your actual property type field
  if(isset($listing->property_type)){
      $type = strtolower($listing->property_type);
      $isCommercial = str_contains($type, 'commercial');
  }
@endphp

@if($isCommercial && !empty($listing->owner_responsible_for))
  <div>
    <strong>Owner Responsible For:</strong>
    {{ $listing->owner_responsible_for }}
  </div>
@endif
```
