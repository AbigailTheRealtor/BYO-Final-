# 3) MODEL SAFEGUARD (PREVENT SILENT FAILURES)

In the Eloquent model that backs Hire a Seller’s Agent listings:

### 3.1 Add to `$fillable`

Add:

* `has_pool`

### 3.2 Add to `$casts`

```php
protected $casts = [
    'has_pool' => 'boolean',
];
```

✅ Safeguard:

* If `has_pool` is not fillable, it can silently fail saving.
* If not cast, it may not repopulate correctly in edit forms.
