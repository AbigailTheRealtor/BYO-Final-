Locations where this must apply:

Problem

Seller “Broker Compensation & Agency Agreement Terms” is currently displaying descriptor text instead of the actual selected numeric value, e.g.:

Incorrect (current):
Seller’s Broker Purchase Fee: Percentage of the Total Purchase Price
Purchase Percentage: 6%

Correct (required):
Seller’s Broker Purchase Fee: 6%
(then keep the detail line only when needed, or omit if redundant)

This same issue repeats across Purchase, Buyer’s Broker comp, Lease, Lease-Option, and Legal Terms.

Required Output Rules (Landlord-Style)
Rule 1 — If a numeric value exists, show it on the main line

For any field where the user selected a type (Percentage / Flat Fee / etc.) AND entered a value:

The main display line must render as:
<Field Label>: <Value>
Examples of value formats:

6%

$5,000

30 days

Yes / No

✅ This means: replace “Percentage of …” / “Flat Fee” / “Commission Structure…” text with the actual value when available.

Rule 2 — Suppress redundant “type description” lines when value is already shown

If the numeric value is displayed on the main line, do NOT also display:

“Percentage of the Total Purchase Price”

“Percentage of the Gross Lease Value”

other type-description-only strings

Those descriptors are acceptable only if there is no numeric value available (edge case).

Rule 3 — Show sub-lines only when they add new information

Some sections have sub-fields that are meaningful and should remain as bullets/indented lines only when applicable, such as:

Lease-Option having two different compensation events
(Option created vs option exercised)

Early Termination Fee = Yes → show Termination Fee Amount

Retainer Fee = Yes → show Retainer Fee Amount and Retainer Fee Application

Do not change the wording of the labels—only the display formatting.

Rule 4 — Keep section headers and spacing exactly as current Landlord-style

Seller Broker Compensation display should mirror the Landlord formatting with:

clear section headers (e.g., “Seller’s Broker Compensation,” “Buyer’s Broker Compensation,” etc.)

clean spacing

bullets/indent only where needed
