# CRITICAL DETAIL ‚Äî Is Property Condition multi-select or single-select?

### If SINGLE-select

* hydrate as string

### If MULTI-select (very common)

‚úÖ Required:

* store as JSON array
* hydrate as array
* Livewire property must be an array
* the UI must expect an array

**Safeguard**
If your DB currently stores a string (comma-separated), fix one of these:

  ‚Ä¶but **prefer JSON** for reliability.
