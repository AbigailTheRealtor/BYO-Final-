n the Hire a Buyer’s Agent Livewire view (the one that renders the tab wizard), I need Income Property to reuse the Residential layouts starting at Tab 3:

Tab 3 – Purchasing Terms

Right now Income is entering the “full service” block (we see DEBUG: FULL_SERVICE BLOCK ENTERED), but nothing inside it renders, so the tabs are blank.

Step 1 – Normalize property type for tabs 3–7

At the top of the Blade view (inside the Livewire component template), add:

@php
    $rawType = $listing->property_type ?? $property_type ?? null;

    // Treat Income like Residential for steps 3+
    $isResidentialOrIncome = in_array($rawType, [
        'residential',
        'income',
        'income_property',
        'income_property_4_units',
        'income_4_units',
    ]);
@endphp

Step 2 – For each tab from 3 onward, use $isResidentialOrIncome

Wherever the tab content is rendered, update the property-type condition so Income uses the same partial as Residential.

Tab 3 – Purchasing Terms
{{-- BEFORE (example) --}}
@if ($listing->property_type === 'residential')
    @include('buyer.tabs.residential-purchasing-terms')
@elseif ($listing->property_type === 'commercial')

@elseif ($listing->property_type === 'business')

{{-- AFTER – treat Income same as Residential --}}
@if ($isResidentialOrIncome)
