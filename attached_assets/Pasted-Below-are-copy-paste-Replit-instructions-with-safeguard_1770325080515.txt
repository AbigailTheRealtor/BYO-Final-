### Location (Tenant listing “show”)

Find the Tenant listing show Blade file and the section where it currently prints:

It will be in one of these common places (your project will match one):

* `resources/views/livewire/tenant-agent-auction-show.blade.php`
* or a shared card partial used by Tenant listings

**Replace ONLY the condition output** with the same chip rendering approach used for Styles.

#### Add a safe “normalize to array” block directly above the Condition output:

```php
@php
  $rawConditions = $auction->acceptable_property_condition ?? null;

  // Normalize to array (supports JSON array, PHP array, or comma-separated string)
  if (is_array($rawConditions)) {
      $conditions = $rawConditions;
  } else {
      $decoded = is_string($rawConditions) ? json_decode($rawConditions, true) : null;

      if (is_array($decoded)) {
          $conditions = $decoded;
      } else {
          $conditions = is_string($rawConditions)
              ? array_filter(array_map('trim', explode(',', $rawConditions)))
              : [];
      }
  }
@endphp
```

#### Then render the chips using the SAME markup/classes as Property Styles:

```php
@if(!empty($conditions))
  <div class="mt-1">
    @foreach($conditions as $condition)
      {{-- IMPORTANT: paste the exact same <span> chip markup/classes used for Acceptable Property Styles --}}
      <span class="YOUR_STYLE_CHIP_CLASSES_HERE">{{ $condition }}</span>
    @endforeach
  </div>
@else
  <span class="text-gray-500">—</span>
@endif
```

✅ **Critical:** “YOUR_STYLE_CHIP_CLASSES_HERE” must be the **exact** classes used by **Acceptable Property Styles** so both look identical.
