#1002 - Direct Hire Counter Terms (Comp Terms Page) Wording & Placeholder Standardization

Example QA URL:
https://3ce01d2c-f79e-44c2-b048-2e8dc64316ef-00-tu7aronbf5wk.spock.replit.dev/hire/agent/direct/142/buyer/residential/counter

What & Why
The Direct Hire Counter Terms → Broker Compensation & Agency Agreement Terms page (“Comp Terms” page) contains inconsistent wording and placeholder formatting.

Current issues include:
- Generic “Agreement Timeframe” label
- “Other (Custom)” option text
- Placeholders using “Specify…”
- Bare “Amount”
- Placeholder-only examples like “e.g. 90”

These should follow the platform-wide wording standard.

Scope
ONLY the Direct Hire Counter Terms → Comp Terms page/section:
- /hire/agent/direct/{id}/{role}/{propertyType}/counter

Apply to all Direct Hire roles:
- Buyer
- Seller
- Tenant
- Landlord

Do NOT change:
- Services section/page
- Additional Terms section/page
- Client Details section/page
- Marketplace hire-agent counter flows
- Routes
- Controllers
- Validation
- Database schema
- Save logic
- Stored values/meta keys

Required Changes

1. Make Agreement Timeframe label role-specific
Replace the generic:
- Agreement Timeframe

With:
- Buyer Agency Agreement Timeframe:
- Seller Agency Agreement Timeframe:
- Tenant Agency Agreement Timeframe:
- Landlord Agency Agreement Timeframe:

Use the existing role context already available in the page/component.

2. Change dropdown option text
Current:
- Other (Custom)

Update to:
- Other

Only for the Agreement Timeframe dropdown on the Comp Terms page.

3. Standardize all placeholders on the Comp Terms page
All placeholders should follow:

Enter [field title] (e.g., Example)

Rules:
- Must start with “Enter”
- Must include the field title or clear purpose
- Must use “(e.g., Example)” with comma after e.g.
- Capitalize example words appropriately
- Do not use “Specify”
- Do not use bare “Amount”
- Do not use placeholder-only examples like “e.g. 90”

Examples:
- Enter agreement timeframe (e.g., 45 Days)
- Enter protection period (e.g., 90 Days)
- Enter early termination fee amount (e.g., 2000)
- Enter retainer fee amount (e.g., 500)

Audit the rest of the Comp Terms page for any inconsistent placeholders and standardize them to this format.

Done looks like
- Buyer page shows:
  Buyer Agency Agreement Timeframe:
- Dropdown option says:
  Other
- All Comp Terms placeholders follow:
  Enter [field title] (e.g., Example)
- Existing values still preload correctly
- Counter terms still save and submit normally
- No console errors
- No layout regressions
