Task #565D — Fix Select2 Placeholder Not Rendering Across Create Offer Listing Flows

ISSUE:
Select2 fields are rendering with icons but NO visible “Select” placeholder text.

This is happening across multiple fields in all Create Offer Listing flows (Seller, Buyer, Landlord, Tenant).

This is NOT a Blade issue and NOT an option issue — this is a Select2 initialization problem.

---

CONFIRMED BROKEN EXAMPLES:

Seller:
- Special Sale Provision
- Offered Financing/Currency

Buyer:
- Acceptable Special Sale Provisions
- Offered Financing/Currency

Landlord:
- Appliances Included
- View
- Desired Lease Term
- Rent Includes
- Tenant Pays

Tenant:
- Offered Lease Term
- Leasing Space

These are examples only. Fix ALL fields that show the same pattern:
👉 icon visible + empty field + NO “Select” text

---

ROOT CAUSE:
Select2 is being initialized inconsistently:
- Some fields are missing placeholder config
- Some are initialized with incorrect $(this) context
- Some are reinitialized by Livewire without placeholder
- Some use different init patterns

Result:
Select2 renders but placeholder is never applied.

FIX — STANDARDIZE SELECT2 INITIALIZATION

In ALL Create Offer Listing main blade files:

- resources/views/livewire/offer-listing/seller/offer-seller-listing.blade.php
- resources/views/livewire/offer-listing/buyer/offer-buyer-listing.blade.php
- resources/views/livewire/offer-listing/landlord/offer-landlord-listing.blade.php
- resources/views/livewire/offer-listing/tenant/offer-tenant-listing.blade.php

---

STEP 1 — CREATE SHARED INITIALIZER

Add this function:

function initSelect2Fields() {
    $('.select2-multiple, .select2-single').each(function () {
        const $el = $(this);

        // Prevent double initialization
        if ($el.hasClass('select2-hidden-accessible')) return;

        const placeholder = $el.attr('data-placeholder') || 'Select';

        $el.select2({
            placeholder: placeholder,
            allowClear: true,
            width: '100%'
        });
    });
}

---

STEP 2 — RUN ON PAGE LOAD

$(document).ready(function () {
    initSelect2Fields();
});

---

STEP 3 — LIVEWIRE RE-INIT (CRITICAL)

document.addEventListener('livewire:load', function () {
    Livewire.hook('message.processed', () => {
        initSelect2Fields();
    });
});

---

IMPORTANT RULES:

- DO NOT use $(this).data('placeholder') outside a valid .each()
- DO NOT initialize Select2 individually per field unless absolutely required
- DO NOT overwrite existing custom configs (merge if needed)
- DO NOT remove existing wire:ignore wrappers

---

MULTI-SELECT RULE (ALREADY FIXED, DO NOT REINTRODUCE):

- Must NOT contain:
  <option value="">Select</option>
  OR
  <option value=""></option>

- Must use:
  data-placeholder="Select"
