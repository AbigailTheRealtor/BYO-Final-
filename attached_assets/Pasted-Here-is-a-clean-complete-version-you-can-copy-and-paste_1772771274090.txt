# FINAL LISTING DISPLAY RULES

The final listing must display cleanly.

## SHOW FILLED DATA ONLY

Fields must display **only if they contain data**.

Examples:

Text fields display only if text exists.
Select fields display only if an option is selected.
Multi-select fields display only if values exist.

---

## HIDE EMPTY FIELDS

Do NOT render empty fields like:

```
View:
Garage/Parking:
Amenities:
```

If a field has no value, the field must not display.

---

## HIDE EMPTY SECTION TITLES

If an entire section contains no filled fields:

The section title and divider must not display.

Example:

If no amenities are selected, do not show:

```
Amenities
---------
```

The entire section should be hidden.

---

## MULTI-SELECT DISPLAY

Multi-select values should display clearly.

Example:

```
Garage/Parking: Covered Parking, Assigned Parking
```

---

## CONDITIONAL SECTIONS

Conditional sections should only appear when relevant.

Examples:

* Lease option details appear only if lease option is selected.
* Commercial sections appear only on commercial listings.

---

## MEDIA DISPLAY

Media should display only if present.

Rules:

* Photos display if uploaded.
* Video links embed if supported.
* If no media exists, the media section should not appear.
