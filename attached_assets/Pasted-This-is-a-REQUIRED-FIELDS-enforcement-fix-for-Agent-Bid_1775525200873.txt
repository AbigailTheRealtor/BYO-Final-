## GOAL

For agent bids, any field marked with a **red star / required indicator** must be required before the agent can submit the bid.

Right now agent bids are submitting even when required fields are missing.

This must be fixed so agent bids behave like listing creation:

* block submit
* show the required fields in red
* show the same style of validation/required-field feedback as listing creation

---

## EXPECTED BEHAVIOR

When an agent clicks **Submit Bid** and required fields are missing:

1. Submission must be blocked
2. Missing required fields must be highlighted in red
3. The validation UX should match listing creation as closely as possible
4. If listing creation shows a top error message / red required state, use that same pattern
5. This must work across all bid tabs, not just the current visible tab

---

All agent bid flows:

* Seller
