State Tooltip

Select the state where the property is located.

B) Buyer + Tenant — Acceptable Cities / Acceptable Counties / Acceptable State
Acceptable Cities Tooltip

Enter the cities where the Buyer/Tenant is willing to purchase/lease a property.
Selecting a city will automatically populate the associated county and state when available.

Buyer version wording: “Buyer … purchase”
Tenant version wording: “Tenant … lease”

Acceptable Counties Tooltip

Enter the counties where the Buyer/Tenant is willing to purchase/lease a property.
If a county is selected, the state will automatically populate.

Buyer version wording: “Buyer … purchase”
Tenant version wording: “Tenant … lease”

Acceptable State Tooltip

Select the state where the Buyer/Tenant is looking to purchase/lease a property.
This may be automatically filled based on the counties selected.

Buyer version wording: “Buyer … purchase”
Tenant version wording: “Tenant … lease”
