## Step 3 — Add wire:key to every conditional block that contains form fields

This is the “stop Livewire from mismatching nodes” fix.

Where you have:

```blade
@if($property_type === 'Residential')
   ... residential fields ...
@endif

@if($property_type === 'Commercial')
   ... commercial fields ...
@endif
```

Change to:

```blade
<div wire:key="tenant-property-details-{{ $property_type }}">
  @if($property_type === 'Residential')
     <div wire:key="tenant-residential-fields">
        ... fields ...
     </div>
  @endif

  @if($property_type === 'Commercial')
     <div wire:key="tenant-commercial-fields">
        ... fields ...
     </div>
  @endif
</div>
```

Do the same for Buyer/Seller/Landlord/Tenant.

**Why this matters:** without wire:key, morphdom reuses nodes in weird ways and can “carry over” hidden classes/styles from a Select2-enhanced select onto a normal select.
