## 5) STOP THE FLASH: Validate activeTab in BOTH Livewire components

In:

* `app/Http/Livewire/LandlordListingShow.php` (or equivalent)
* `app/Http/Livewire/SellerListingShow.php` (or equivalent)

Add this method (or equivalent) to prevent invalid tabs from rendering briefly:

```php
public function updatedActiveTab($value)
{
    $allowed = array_keys($this->tabs ?? []);

    // If tabs is not associative, use:
    // $allowed = $this->tabs ?? [];

    if (!in_array($value, $allowed, true)) {
        $this->activeTab = $allowed[0] ?? 'details';
    }
}
```

Also ensure you have a safe default:

```php
public $activeTab = 'details';

public function mount()
{
    if (!$this->activeTab) {
        $this->activeTab = 'details';
    }
}
```
