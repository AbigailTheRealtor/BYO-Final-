For Hire a Seller’s Agent listings, the live listing page must display the exact same field labels/verbiage and field order that appear during Seller listing creation.

Do not rewrite or invent labels.
Do not use Tenant/Buyer label sets.
Instead: locate the exact verbiage used on the Seller creation tabs and reuse it for the live Seller listing display.

But the live Seller listing output is using a different renderer/label set (Tenant-style wording and/or wrong sections), so:

labels don’t match creation

some fields are missing

property type specific fields (Income/Commercial/Business/Vacant Land) don’t match creation
