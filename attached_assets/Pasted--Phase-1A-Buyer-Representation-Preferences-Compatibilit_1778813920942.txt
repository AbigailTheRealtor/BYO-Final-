# Phase 1A — Buyer Representation Preferences & Compatibility Tab

What & Why
Add a new "Representation Preferences & Compatibility" tab to the Hire Buyer’s Agent wizard (Full Service flow only). This is a structured compatibility questionnaire that lets Buyers describe their communication preferences, negotiation style, decision-making approach, risk tolerance, support expectations, and ideal Agent working style.

This supports BidYourAgent’s premium representation marketplace direction by helping Agents better understand the Buyer’s goals and working preferences before submitting representation proposals.

This task is additive-only. No existing tabs, fields, logic, routes, migrations, or unrelated flows may be modified.

Done looks like
- A new "Representation Preferences & Compatibility" tab appears in the Full Service Hire Buyer Agent wizard.
- The tab is positioned immediately before the existing "Broker Compensation & Agency Agreement Terms" tab.
- The tab only appears when user_type = buyer (or equivalent condition in the component).
- All fields render correctly with matching platform styles, spacing, icons, placeholders, and tooltips.
- Multi-select fields use Select2 with closeOnSelect: false and placeholder "Select".
- Single-select fields use Select2 with placeholder "Select".
- Required fields validate on Submit / Save Edit only, not Save Draft.
- All fields persist on Save Draft and reload correctly when drafts are reopened.
- All fields persist on Save Edit and reload correctly in edit mode.
- Older drafts/listings without compatibility_preferences still open and save normally.
- Existing fields are not erased; unrelated meta values are not overwritten.
- Next/Back buttons, correction mode, mobile wizard behavior, and Submit/Finish placement remain correct.
- The Buyer Information / Review tab shows a "Representation Preferences & Compatibility" section displaying only non-empty values.
- No Seller, Landlord, Tenant, Offer Listing, Direct Hire, or Limited Service files are changed.

Out of scope
- AI scoring
- match percentages
- ranking/recommendation logic
- Offer Listing flows
- Direct Hire flows
- Limited Service / Flat Fee flows
- schema migrations
- route changes
