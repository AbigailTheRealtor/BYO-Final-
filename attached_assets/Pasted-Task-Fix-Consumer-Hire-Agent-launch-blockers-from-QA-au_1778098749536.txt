- bidding period auto-transition
- expiry guard on accept/reject

Known affected fields:

- auction_type
- expiration_date

Known affected areas:

- saveSABid()

- acceptSABid()
- rejectSABid()

Task:

  - Prefer $model->getMeta('field_name') if available
  - Otherwise use the established $model->info('field_name') pattern if that is what the model supports
- Do not change field names or meta keys.
- Do not rewrite controller methods.
- Do not change bidding rules beyond restoring the existing intended logic.

Specific intended behavior:
- property_type should read the actual seller listing property type, not always fall back to residential.
- auction_type should read the actual seller listing type.
- expiration_date should read the actual seller listing expiration date.
- Expired seller listings should not allow bid accept/reject when existing logic says they should be blocked.
- Bidding Period auto-transition should evaluate using real auction_type and expiration_date values.
