2) Add local format helpers (ONLY inside this view or its partial)

At the top of the bid display loop (or immediately before the compensation section), add:

@php
  $fmtMoney0 = function($v) {
    if ($v === null || $v === '') return null;
    $raw = preg_replace('/[^0-9.]/', '', (string)$v);
    if ($raw === '' || !is_numeric($raw)) return null;
    return '$' . number_format((float)$raw, 0);
  };

  $fmtPercent = function($v) {
    if ($v === null || $v === '') return null;
    $raw = preg_replace('/[^0-9.]/', '', (string)$v);
    if ($raw === '' || !is_numeric($raw)) return null;
    $n = (float)$raw;
    return (floor($n) == $n ? (string)(int)$n : (string)$n) . '%';
  };

  $joinPlus = function($parts) {
    $parts = array_values(array_filter($parts, fn($p) => $p !== null && $p !== ''));
    return count($parts) ? implode(' + ', $parts) : null;
  };

  $ofBasis = function($basis) {
    return $basis ? ('of ' . $basis) : null;
  };
@endphp


✅ This matches how the “private bids” format is shown and avoids the scaling bug (no dividing).
