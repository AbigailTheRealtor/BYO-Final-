BASELINE RULE (MANDATORY)

Match comparison is always against the other party’s latest version.

Baseline selection:

Tenant viewing Agent Bid / Counter

Agent viewing Tenant Counter

Baseline = Agent’s most recent bid/counter

This baseline applies only to fields inside Broker Compensation & Agency Agreement Terms and Offered Services.

MATCH DEFINITION (STRICT)

A field is a MATCH if:

Current value === baseline value

Field was not edited away from the baseline

A field is NOT A MATCH if:

Any difference exists from baseline

Field was manually edited

There is no partial match logic.

NORMALIZATION RULES

Before comparison:

Trim whitespace

Normalize numeric formatting

Compare value and selected type (e.g., percent vs flat fee)

Services are compared as booleans (included / not included)

SCORING LOGIC
Broker Compensation & Agency Agreement Terms

Score = matched fields ÷ total comparable fields × 100

Offered Services

Score = matched services ÷ total services × 100

Total Match Score

Average of the two section scores

If only one section exists, total = that section’s score
