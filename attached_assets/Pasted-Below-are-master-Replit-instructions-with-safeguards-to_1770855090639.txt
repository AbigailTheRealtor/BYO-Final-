## Implementation (Do This Exactly)

### A) Store services as an ordered array at save time

When user checks services:

* Save `selected_services` as an array in the order they appear in the UI (top to bottom).

1. having a “master ordered list” array for that property type

### B) Listing display must render using the master list order

On listing display:

* Loop through the master ordered list
