What & Why

The .input-group-text-seller CSS block inside the Landlord Livewire root blade diverged from the Seller version. As a result, the $ prepend on Landlord currency fields looks different from Seller:
- wrong vertical padding
- wrong background shade
- missing right border
- only left corners rounded
- input text still gets 50px left padding because the sibling-selector override is missing

Task #715 already aligned the Blade markup correctly. This is now strictly a CSS parity fix only.
