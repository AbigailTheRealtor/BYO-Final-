# 3) Draft Save must persist `offered_financing_details` (this is where Replit usually fails)

## Step 3.2 — Ensure BOTH are saved:

* `offered_financing`
* `offered_financing_details`

### ✅ REQUIRED: Merge safeguard (prevents wiping saved values)

If the UI hides some sections at save time, a plain assignment can overwrite existing JSON with empty arrays.

Use:

```php
$existing = (array) ($model->offered_financing_details ?? []);
$current  = (array) ($this->offered_financing_details ?? []);

$merged = array_replace_recursive($existing, $current);

$model->offered_financing = $this->offered_financing;
$model->offered_financing_details = $merged;
$model->save();
```

⚠️ Safeguard: DO NOT do `$model->offered_financing_details = $this->offered_financing_details;` unless you are 100% sure the array is always complete.
