# STEP 2 — Livewire: Remove ADU from state when switching to Commercial

### Location (Tenant Livewire Create + Edit)

* `app/Http/Livewire/TenantAgentAuction.php`
* `app/Http/Livewire/TenantAgentAuctionEdit.php`

Add/update the property type watcher to call the ADU cleanup.

If your bound property is `property_type`:

```php
public function updatedProperty_type($value)
{
    $this->enforceLeasingSpaceByPropertyType($value);
}
```

(or `updatedPropertyType` if that’s your naming)

Add this method:

```php
private function enforceLeasingSpaceByPropertyType($value): void
{
    $adu = 'Accessory Unit / Guest Suite (ADU)';

    // Only enforce removal when NOT Residential
    if ($value !== 'Residential') {

        // Single select
        if (is_string($this->leasing_space ?? null)) {
            if ($this->leasing_space === $adu) {
                $this->leasing_space = null;
            }
            return;
        }

        // Multi select
        if (is_array($this->leasing_space ?? null)) {
            $this->leasing_space = array_values(array_filter(
                $this->leasing_space,
                fn($v) => $v !== $adu
            ));
        }
    }
}
```

✅ **Safeguard:** removes only ADU, keeps valid Commercial leasing space selections.
