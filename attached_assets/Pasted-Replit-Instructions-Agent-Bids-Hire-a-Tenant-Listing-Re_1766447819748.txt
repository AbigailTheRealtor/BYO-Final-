Problem

On Agent Presentation & Marketing Materials, when the user clicks + Add Social Media, a browser validation bubble appears: “Please enter a number.” This should not happen.

Required Fix

Locate the Social Media repeater/row that gets added when clicking + Add Social Media.

One of the inputs in that row is incorrectly set to type="number" or has numeric validation (min, step, numeric rules) causing the browser’s built-in message.

Change that field to a non-numeric type (usually type="text" or type="url") OR remove the HTML numeric constraint that triggers the bubble.

Ensure the “Add Social Media” button does not trigger form submission:

If it’s a <button> inside a form, make sure it is type="button" (not submit).
