I like **Listing ID first, then Status**.

Reason:

* **Listing ID** is more like the permanent reference number
* **Status** is the changing state
* visually it feels cleaner to show:

  1. title
  2. listing ID
  3. status
* it also matches how Seller, Buyer, and Landlord already look, so I would standardize Tenant to that same order

For colors, I would recommend:

* **Active** = green
* **Pending** = amber / orange
* **Hired Agent** = blue
* **Expired** = gray

That gives each status a distinct meaning:

* green = open/live
* amber = paused/in review
* blue = completed/selected
* gray = closed/inactive
