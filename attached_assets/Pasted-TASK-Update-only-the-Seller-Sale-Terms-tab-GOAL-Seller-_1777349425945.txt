Seller Sale Terms will collect the Seller’s baseline purchase terms for the future BidYourOffer “Bid Now” flow.

SELLER SALE TERMS TAB:

Helper Notice:
Enter the Seller’s preferred sale terms. These terms will be used as the starting point for Buyer offers and future counteroffers.

Keep these existing fields:
1. Special Sale Provision
2. Target Closing Timeframe
3. Occupant Type
4. Desired Sale Price
5. Offered Financing/Currency

Add these fields:

1. Initial Deposit Requested
Input type: money
Placeholder: Enter requested initial deposit amount (e.g., 10000)

2. Initial Deposit Due Timeframe
Dropdown options:
- Upon Acceptance
- Within 1 Day After Acceptance
- Within 3 Days After Acceptance
- Within 5 Days After Acceptance
- Other

3. Additional Deposit Requested
Input type: money
Placeholder: Enter additional deposit amount, if any (e.g., 15000)

4. Additional Deposit Due Timeframe
Dropdown options:
- Not Applicable
- Within 3 Days After Acceptance
- Within 5 Days After Acceptance
- Within 10 Days After Acceptance
- After Inspection Period
- Other

5. Escrow Agent / Deposit Holder Preference
Input type: text
Placeholder: Enter preferred escrow agent, title company, attorney, or brokerage, if any.

6. Preferred Inspection Period
Input type: number
Placeholder: Enter preferred inspection period in days (e.g., 10)

7. Appraisal Contingency Preference
Dropdown options:
- Acceptable
- Not Preferred
- To Be Negotiated
- Not Applicable

8. Financing Contingency Preference
Dropdown options:
- Acceptable
- Not Preferred
- To Be Negotiated
- Not Applicable

9. Sale of Buyer Property Contingency Preference
Dropdown options:
- Not Accepted
- Will Consider
- To Be Negotiated

10. Seller Contribution / Credit Offered
Dropdown options:
- None
- Seller Will Consider Buyer Request
- Fixed Dollar Amount
- Percentage of Purchase Price
- Seller to Pay Specific Costs
- Other

11. Seller Contribution Amount / Details
Show only if Seller Contribution / Credit Offered is not “None”.
Input type: text
Placeholder: Enter contribution amount, percentage, or specific costs Seller may consider.

12. Possession Preference
Dropdown options:
- At Closing
- After Closing
- Seller Leaseback Requested
- Subject to Tenant Occupancy
- Other

13. Possession Details
Show only if Possession Preference is not “At Closing”.
Input type: text
Placeholder: Enter possession timing, leaseback terms, tenant occupancy details, or other possession conditions.

14. Included Personal Property / Fixtures
Input type: textarea
Placeholder: List any appliances, fixtures, equipment, or personal property included in the sale.

15. Excluded Items
Input type: textarea
Placeholder: List any items excluded from the sale.

16. Home Warranty Offered
Dropdown options:
- No
- Yes, Seller Will Provide
- Seller Will Consider Buyer Request
- To Be Negotiated

17. Home Warranty Amount / Details
Show only if Home Warranty Offered is not “No”.
Input type: text
Placeholder: Enter home warranty amount, provider preference, or related details.

18. HOA / Condo / Association Terms
Input type: textarea
Placeholder: Enter any HOA, condo, association approval, document, fee, or transfer terms.

19. Additional Seller Sale Terms
Input type: textarea
Placeholder: Enter any additional sale terms the Seller wants buyers to consider.

SAVE REQUIREMENTS:
- Add these fields to the Seller Sale Terms Livewire state.
- Save them using the current listing save pattern.
- Include them in save draft.
- Include them when editing an existing Seller listing.
- Make new fields optional unless already required by existing logic.
- Money fields should be nullable numeric decimals.
- Number fields should be nullable numeric.
- Text and textarea fields should be nullable strings.
- Dropdown fields should validate only against the listed options.
