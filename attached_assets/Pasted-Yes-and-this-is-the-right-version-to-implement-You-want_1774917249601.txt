A. BASELINE RULE
Use only the Tenant’s original listing values as the source of truth.

Baseline includes:

1. Tenant’s originally selected services
2. Tenant’s original Additional Services entries
3. Tenant’s originally filled Broker Compensation & Agency Agreement Terms fields

Do not use Agent-added values to increase the score.

B. SERVICES MATCH LOGIC

1. Each Tenant original selected service = 1 baseline service point
2. Each Tenant original Additional Service entry = 1 baseline service point
3. If the Agent still offers that same baseline item, classify as Matched
4. If the Agent does not offer that original baseline item, classify as Missing
5. If the Agent adds a service that was not part of the Tenant baseline, classify as Extra (not in baseline)
6. Extra services must not increase the numerator or denominator

services_match_percent = matched_services / services_baseline_total * 100

Important:

7. Changed fields remain in the denominator because they were part of the Tenant baseline

terms_match_percent = matched_terms / terms_baseline_total * 100
