## Step 1 Fix the Buyer EDIT route so Buyer edits use BuyerAgentAuctionEdit, not TenantAgentAuctionEdit

In routes file where this exists:

### Option A Best

Create separate routes so Buyer does not share Tenant edit route.

Example pattern
