## C2 — Normalize selected services into a flat array BEFORE building snapshot (required)

Add a helper:

```php
private function normalizeSelectedServices($raw): array
{
    if (is_null($raw)) return [];

    // If already array of strings
    if (is_array($raw)) {
        // Flatten any nested arrays
        $flat = [];
        array_walk_recursive($raw, function ($v) use (&$flat) {
            if (is_string($v) && trim($v) !== '') $flat[] = trim($v);
        });
        return array_values(array_unique($flat));
    }

    // JSON string
    if (is_string($raw)) {
        $decoded = json_decode($raw, true);
        if (is_array($decoded)) {
            return $this->normalizeSelectedServices($decoded);
        }

        // Comma-separated fallback
        return array_values(array_unique(array_filter(array_map('trim', explode(',', $raw)))));
    }

    return [];
}
```

Then in snapshot build:

```php
$selectedRaw = $this->services_selected ?? $this->services ?? $this->services_requested ?? null;
$selected = $this->normalizeSelectedServices($selectedRaw);
```

✅ This prevents missing items due to nested arrays / JSON strings / commas.

## C4 — Matching must be exact, but must also handle invisible character differences (smart-safe)

Sometimes the UI list uses “smart quotes” and stored values use straight quotes, so exact comparison fails.

Add a safe comparator just for matching (Tenant-only; does not change displayed text):

```php
private function canon(string $s): string
{
    $s = trim($s);

    // normalize curly quotes to straight quotes for matching only
    $s = str_replace(["’","“","”"], ["'","\"","\""], $s);

    // collapse whitespace
    $s = preg_replace('/\s+/', ' ', $s);

    return $s;
}
```

Then when checking:

```php
$selectedCanon = array_map(fn($x) => $this->canon($x), $selected);

foreach ($catalog as $section => $items) {
  foreach ($items as $text) {
     $checked = in_array($this->canon($text), $selectedCanon, true);
     ...
  }
}
```

✅ **Safeguard:** This does not change UI wording; only improves matching accuracy.

This specifically fixes items like:

* Craigslist sections with quotes
* Smart apostrophes in “Tenant’s”
* Curly quotes around “Real Estate Wanted”
