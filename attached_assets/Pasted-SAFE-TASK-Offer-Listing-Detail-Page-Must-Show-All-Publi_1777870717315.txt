SAFE TASK — Offer Listing Detail Page Must Show All Public Fields from All Tabs

Issue:
When a user submits an Offer Listing, the resulting detail/view page currently shows only a small subset of the data entered.

Goal:
The submitted Offer Listing detail/view page must display all public-facing saved fields from the current create/edit tabs, organized in the same section order, similar to Hire Agent listing detail pages.

Applies to:
- Seller
- Buyer
- Landlord
- Tenant
- All property/user types

Important rule:
AI Questions should NOT display on the listing detail page.
AI Questions must still be saved for future chatbot/Brainlee use, but they are internal/private and should not appear publicly.

Show on detail page:
- Listing Details
- Property Details / Property Preferences
- Sale Terms / Purchase Terms / Leasing Terms
- Financial Details where applicable
- Additional Details
- Tax, Legal, HOA & Disclosures where applicable
- Photos, Tours & Documents
- Agent Credentials & Contact Info

Do NOT show on detail page:
- AI Questions
- Internal-only chatbot training prompts
- Hidden/admin-only metadata

Required Fix:
1. Trace all current create/edit tabs and fields for Seller, Buyer, Landlord, and Tenant.
2. Confirm every public-facing field is saved on Submit.
3. Update submitted Offer Listing detail/view pages to display all saved public fields.
4. Organize display sections in the same order as the create/edit flow.
5. Hide optional empty fields unless the current Hire Agent listing detail style intentionally shows them.
6. Display uploaded photos, video/3D tour links, and documents where applicable.
7. Keep AI Questions saved but excluded from the listing detail page.

Do NOT:
- Change database schema unless absolutely required and reported first
- Touch Hire Agent flows
- Touch Direct Hire counter forms
- Change field names
- Change validation rules unless required for save/display parity
- Display AI Questions publicly

Verification:
For Seller, Buyer, Landlord, and Tenant:
- Create a listing.
- Fill fields across every tab, including AI Questions.
- Submit listing.
- Open detail/view page.
- Confirm all public fields display in correct section order.
- Confirm AI Questions do NOT display.
- Confirm AI Questions are still saved in the database/meta for future chatbot use.
- Confirm uploaded photos/documents/tour URLs display correctly.
