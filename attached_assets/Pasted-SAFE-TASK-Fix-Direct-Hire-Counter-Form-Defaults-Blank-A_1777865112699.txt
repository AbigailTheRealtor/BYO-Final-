SAFE TASK — Fix Direct Hire Counter Form Defaults: Blank Additional Details + Agent-Selected Services Only

Issue:
After clicking “Request Changes / Counter Terms” from the Direct Hire / Hire Me review page, the editable counter form opens, but:

1. Additional Details is pre-filled with the Agent’s preset/profile text.
2. Offered Services tab does NOT show the Agent’s selected offered services as checked.
3. Offered Services only shows “Other – Specify additional services as needed.”

Correct behavior:
When a client requests changes/counter terms:

Additional Details:
- Must be blank by default for a NEW counter.
- Do NOT auto-fill from Agent preset/profile.
- Do NOT auto-fill from original bid unless editing an already-submitted counter.
- Only show prior text if the user is editing an existing submitted/saved counter.

Offered Services:
- Show ONLY the services the Agent included in their preset/proposal.
- Each Agent-selected service should appear as an editable checked checkbox.
- User may uncheck services they do not need.
- Do NOT show the full recommended service catalog.
- Do NOT show services the Agent did not include.
- Keep Additional Services / Other field for client-requested extra services.

Scope:
- Direct Hire counter flow only
- Counter Terms 3-tab form
- Offered Services tab
- Additional Details tab
- All roles/property types:
  Buyer: Residential, Income, Commercial, Business, Vacant Land
  Seller: Residential, Income, Commercial, Business, Vacant Land
  Landlord: Residential, Commercial
  Tenant: Residential, Commercial

Do NOT:
- Change Agent Preset save logic
- Change service catalog text
- Change Offer Listing flows
- Change accepted summary logic
- Change database schema unless absolutely required and reported first
- Change the read-only Hire Me review page except if needed to pass context correctly

Tasks:
1. Audit current data source for counter form:
   - Where Additional Details is being populated from
   - Where Offered Services options are being populated from
   - Where Agent preset selected services are stored
   - Where Direct Hire bid services are stored

2. Fix Additional Details:
   - For new counter from Direct Hire, initialize as blank.
   - Preserve existing counter text only when editing existing counter terms.

3. Fix Offered Services tab:
   - Load Agent’s selected services from the Direct Hire bid/preset.
   - Render only those selected services.
   - Render them as checked checkboxes.
   - Allow user to uncheck them.
   - Preserve service category grouping if available.
   - Do not include unselected catalog services.

4. Preserve Additional Services:
   - Keep “Other – Specify additional services as needed” or Additional Services textarea for new client-requested services.
   - This should be separate from Agent-selected services.

5. Submit behavior:
   - Submitted counter should save the remaining checked Agent-selected services.
   - Submitted counter should save any Additional Services requested by client.
   - Unchecked Agent services should not be included in the counter.
