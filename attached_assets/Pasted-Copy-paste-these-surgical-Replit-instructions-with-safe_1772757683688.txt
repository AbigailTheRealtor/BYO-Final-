# WHAT’S HAPPENING (THE BUG)

When a user selects an exchange item, it appears briefly then disappears because:

* Livewire re-renders after the change
* Select2 gets destroyed/rebuilt or loses state
* The Livewire property doesn’t reliably receive the array values (or values aren’t rehydrated back into Select2)

### Goal
