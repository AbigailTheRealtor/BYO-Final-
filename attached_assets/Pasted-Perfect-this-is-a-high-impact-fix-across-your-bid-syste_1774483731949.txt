### In Bid Component (Livewire mount/init):

```
public function mount($listingId)
{
    $this->listing = Listing::findOrFail($listingId);

    // ALWAYS override with listing data
    $this->broker_compensation = $this->listing->broker_compensation;
    $this->agency_terms = $this->listing->agency_terms;
    $this->offered_services = $this->listing->offered_services;
}
```

---

### 🔒 HARD OVERRIDE (prevents wrong data bleed)

```
public function hydrateFromListing()
{
    if ($this->listing) {
        $this->broker_compensation = $this->listing->broker_compensation;
        $this->agency_terms = $this->listing->agency_terms;
        $this->offered_services = $this->listing->offered_services;
    }
}
```

Call this:

```
$this->hydrateFromListing();
```
