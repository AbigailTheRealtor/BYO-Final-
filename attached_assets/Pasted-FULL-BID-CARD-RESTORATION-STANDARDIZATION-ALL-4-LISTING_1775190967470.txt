REQUIRED UNIVERSAL BID CARD STRUCTURE
==================================================

- Agent label/name
- status badge

2. Offered Services line

3. Extra Value Added line
Format:
Extra Value Added: X Services
— does not affect match score

4. Match Score box
This must be restored on all 4 roles.

The Match Score box must include:
- Match Score heading
- overall percentage badge
- Services Match %
- Matched: X/Y
- Terms Match %
- Matched: X/Y
- Compared to: [correct baseline label]
- explanatory note about added services/terms not increasing score

5. Broker Compensation Summary

6. View Full Bid

7. Edit Bid
- only when allowed for the bidding agent
- keep current permission logic where appropriate

==================================================
MATCH SCORE BOX FORMAT
==================================================

Restore the Match Score box using the structure already approved:

- Match Score
