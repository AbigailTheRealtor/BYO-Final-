# STEP 1 — Create a reusable Blade component for “Text vs Chips”

Create this file:

**`resources/views/components/value-display.blade.php`**

Paste this:

```blade
@props([
    'value' => null,         // string | array | json
    'normalize' => true,     // normalize wording like "Residential Property" => "Residential"
])

@php
    $raw = $value;

    // Decode JSON strings safely
    if (is_string($raw)) {
        $decoded = json_decode($raw, true);
        if (json_last_error() === JSON_ERROR_NONE) {
            $raw = $decoded;
        }
    }

    // Normalize function (display-only)
    $norm = function ($v) use ($normalize) {
        if (!$normalize || !is_string($v)) return $v;

        $vTrim = trim($v);

        // Remove trailing " Property" in common cases
        // e.g. "Residential Property" => "Residential"
        if (preg_match('/^(Residential|Commercial|Vacant Land|Business Opportunity|Income)\s+Property$/i', $vTrim, $m)) {
            return ucfirst(strtolower($m[1]));
        }

        // Generic fallback: if it ends with " Property", remove that suffix
        if (preg_match('/\s+Property$/i', $vTrim)) {
            return preg_replace('/\s+Property$/i', '', $vTrim);
        }

        return $vTrim;
    };

    // Convert to array of selections if applicable
    $items = [];

    if (is_array($raw)) {
        // Flatten categorized arrays or keep flat arrays
        $isFlat = count($raw) === count(array_filter($raw, 'is_string'));
        if ($isFlat) {
            $items = $raw;
        } else {
            foreach ($raw as $cat => $arr) {
                if (is_array($arr)) {
                    foreach ($arr as $it) $items[] = $it;
                }
            }
        }
    } elseif (is_string($raw) && trim($raw) !== '') {
        $items = [$raw];
    }

    $items = array_values(array_unique(array_filter(array_map(fn($x) => is_string($x) ? trim($x) : $x, $items))));
    $items = array_values(array_filter($items, fn($x) => is_string($x) ? $x !== '' : true));
    $count = count($items);

    // Apply normalization to each item
    $items = array_map(fn($x) => $norm($x), $items);
@endphp

@if($count === 1)
    <span>{{ $items[0] }}</span>
@elseif($count > 1)
    <div class="flex flex-wrap gap-2">
        @foreach($items as $it)
            <span class="px-3 py-1 rounded-full border text-sm">{{ $it }}</span>
        @endforeach
    </div>
@endif
```

This component:

* Accepts string/array/JSON
* Normalizes “Residential Property” → “Residential”
* Shows **text** for 1 value, **chips** for 2+ values
