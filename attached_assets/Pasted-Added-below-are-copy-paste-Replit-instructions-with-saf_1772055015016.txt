### 2B) Add safe decode helper (if not already added)

If you don’t already have one in this blade:

```php
@php
  $decodeToArray = function ($value) {
      if (is_array($value)) return $value;
      if (is_null($value) || $value === '') return [];
      if (is_string($value)) {
          $decoded = json_decode($value, true);
          return is_array($decoded) ? $decoded : [$value];
      }
      return [];
  };
@endphp
```
