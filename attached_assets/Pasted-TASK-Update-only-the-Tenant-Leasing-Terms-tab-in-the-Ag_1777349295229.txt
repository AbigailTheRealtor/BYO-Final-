TASK: Update only the Tenant Leasing Terms tab in the Agent Dashboard Create Listing / Edit Listing flow.

LOCATION:
This applies only to the listing creation/editing workflow inside the Agent Dashboard when creating or editing a Tenant listing.

Do not edit public listing pages, Bid Now pages, agent bid forms, offer preset forms, dashboards, or accepted bid summaries.

GOAL:
Tenant Leasing Terms will collect the Tenant’s baseline lease terms for the future BidYourOffer “Bid Now” flow.

SAFEGUARDS:
- Do not edit Seller, Buyer, or Landlord tabs.
- Do not edit Services, Additional Details, Broker Compensation, Referral & Cooperation, Agent Credentials, dashboards, offer presets, bid cards, bid modals, accepted bid summaries, or public listing pages.
- Do not change tab order.
- Do not remove existing save draft or edit listing behavior.
- Do not touch BidYourAgent bid/counter/accept logic.

TENANT LEASING TERMS TAB:

Helper Notice:
Enter the Tenant’s desired lease terms. These terms will be used as the starting point for Landlord lease offers and future counteroffers.

Keep these existing fields:
1. Maximum Monthly Lease Price
2. Offered Lease Date
3. Leasing Space

Add these fields:

1. Desired Lease Length
Input type: text
Placeholder: Enter desired lease length (e.g., 12 months)

2. Security Deposit Budget
Input type: money
Placeholder: Enter maximum security deposit the Tenant is willing to offer.

3. Move-In Funds Available
Input type: money
Placeholder: Enter total move-in funds available, if applicable.

4. First Month Rent Available
Dropdown options:
- Yes
- No
- Negotiable

5. Last Month Rent Available
Dropdown options:
- Yes
- No
- Negotiable

6. Pet Information
Input type: textarea
Placeholder: Describe pets, breed, weight, number of pets, and any pet-related terms.

7. Number of Occupants
Input type: number
Placeholder: Enter number of occupants.

8. Parking Needed
Input type: textarea
Placeholder: Describe parking needs, assigned spaces, guest parking, or vehicle details.

9. Utility Preference
Input type: textarea
Placeholder: Describe preferred utility responsibility or utilities requested to be included.

10. Maintenance Preference
Input type: textarea
Placeholder: Describe preferred maintenance, lawn, pool, pest control, repair, or service responsibilities.

11. Renewal Option Requested
Dropdown options:
- No
- Yes
- Negotiable

12. Renewal Option Details
Show only if Renewal Option Requested is “Yes” or “Negotiable”.
Input type: textarea
Placeholder: Enter requested renewal option terms.

13. Tenant Conditions
Input type: textarea
Placeholder: Enter any conditions such as application approval, HOA approval, move-in inspection, school timing, job relocation, or other lease conditions.

14. Additional Tenant Lease Terms
Input type: textarea
Placeholder: Enter any additional lease terms the Tenant wants landlords to consider.

COMMERCIAL-ONLY FIELDS:
Show only when property type is commercial.

15. Commercial Lease Type Preference
Dropdown options:
- Gross Lease
- Modified Gross Lease
- Triple Net / NNN
- Full Service Gross
- Percentage Rent
- Other

16. CAM / NNN / Additional Rent Preference
Input type: textarea
Placeholder: Enter any CAM, tax, insurance, operating expense, or additional rent preferences.

17. Rent Escalation Preference
Input type: textarea
Placeholder: Enter preferred annual increases, fixed increases, CPI adjustments, or escalation limits.

18. Buildout / Tenant Improvement Request
Input type: textarea
Placeholder: Describe buildout needs, delivery condition, tenant improvement allowance, or requested work.

19. Intended Business Use
Input type: textarea
Placeholder: Describe intended business use, services offered, products sold, or operational needs.

20. Signage Request
Input type: textarea
Placeholder: Describe requested signage rights or visibility needs.

21. Commercial Parking / Access Needs
Input type: textarea
Placeholder: Describe parking, loading, delivery, truck access, customer access, or employee parking needs.

22. Personal Guarantee Preference
Dropdown options:
- Will Provide
- Will Not Provide
- Negotiable

23. Commercial Approval Conditions
Input type: textarea
Placeholder: Enter any permit, license, franchise, financing, zoning, or corporate approval conditions.

SAVE REQUIREMENTS:
- Add these fields to the Tenant Leasing Terms Livewire state.
- Save them using the current listing save pattern.
- Include them in save draft.
- Include them when editing an existing Tenant listing.
- Make new fields optional unless already required by existing logic.
- Money fields should be nullable numeric decimals.
- Number fields should be nullable numeric.
- Text and textarea fields should be nullable strings.
- Dropdown fields should validate only against the listed options.
- Commercial-only fields should only show for commercial Tenant listings.

VERIFY:
- Tenant Leasing Terms tab loads.
- Tenant listing saves as draft.
- Tenant listing reloads with the new fields pre-filled.
- Commercial-only fields only show for commercial Tenant listings.
- No other role tabs or BidYourAgent logic changed.
