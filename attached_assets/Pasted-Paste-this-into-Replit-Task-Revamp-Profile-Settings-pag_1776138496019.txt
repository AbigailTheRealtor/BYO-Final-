## Task: Revamp Profile Settings page for client users only

This Profile Settings page is for:

* Tenant
* Buyer
* Seller
* Landlord

The goal is to make it clean, simple, and stable for normal users.

This is **not** an agent dashboard settings page.
Keep it lightweight and easy to use.

---

## Part 1 — Fix the accordion bug first

### Current problem

When clicking a Profile Settings section, it flashes open and disappears.

### Required behavior

The settings sections must behave like a stable accordion:

* click once = open
* click again = close
* no flashing
* no disappearing
* no immediate collapse after rerender

### What to inspect

Check whether this page is currently mixing:

* Alpine
* Livewire
* Bootstrap collapse
* duplicate click handlers

### Implementation rule

Use **one accordion system only**.

### Preferred approach

If this page is Livewire-driven:

* use Alpine for the accordion open/close UI state
* prevent Livewire rerenders from resetting the accordion state
* use `wire:ignore`, `wire:key`, or component isolation where needed

Do not leave multiple accordion systems fighting each other.
