Add handler:

```php
private function handleResidentialOnlyTenantFields($value): void
{
    if ($value !== 'Residential') {

        // A) Pool
        $this->pool_type = null;

        // B) Carport
        $this->carport_needed = null;
        $this->carport_spaces_needed = null;

        // C) Leasing Space: remove ADU only (do not clear valid commercial values)
        $this->removeAduFromLeasingSpaceSelection();

        // D) Pets (clear all)
        $this->pets = null; // yes/no (or your actual field)
        $this->number_of_pets = null;
        $this->type_of_pets = null;
        $this->breed_of_pets = null;
        $this->weight_of_pets_lbs = null;
        $this->service_animal = null;
        $this->emotional_support_animal = null;

        // If you have any other pet-related fields (notes, restrictions, etc.), clear them too.
        $this->pet_notes = null; // only if exists
    }
}
```

### Helper to remove ADU only

```php
private function removeAduFromLeasingSpaceSelection(): void
{
    $aduLabel = 'Accessory Unit / Guest Suite (ADU)';

    if (is_string($this->leasing_space ?? null)) {
        if ($this->leasing_space === $aduLabel) {
            $this->leasing_space = null;
        }
        return;
    }

    if (is_array($this->leasing_space ?? null)) {
        $this->leasing_space = array_values(array_filter(
            $this->leasing_space,
            fn($v) => $v !== $aduLabel
        ));
    }
}
```

✅ **Safeguard:** Doesn’t wipe commercial leasing space values—only removes ADU.
