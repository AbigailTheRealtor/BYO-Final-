A2) Enforce Livewire-only navigation + guard (Buyer Edit only)
In Buyer Edit Livewire component

Add (or align to this pattern):

public int $activeTab = 1;
public bool $isNavigating = false;

public function goNext(): void
{
    if ($this->isNavigating) return;
    $this->isNavigating = true;

    // Optional: validate only current tab
    // $this->validateCurrentTab();

    $this->activeTab = min($this->activeTab + 1, $this->maxTab());

    $this->dispatchBrowserEvent('buyer-edit-nav-unlock');
}

public function goBack(): void
{
    if ($this->isNavigating) return;
    $this->isNavigating = true;

    $this->activeTab = max($this->activeTab - 1, 1);

    $this->dispatchBrowserEvent('buyer-edit-nav-unlock');
}

public function maxTab(): int
{
    return 8; // set to your Buyer tab count
}

In Buyer Edit Blade only

Add unlock script:

<script>
  window.addEventListener('buyer-edit-nav-unlock', () => {
    setTimeout(() => {
      @this.set('isNavigating', false);
    }, 50);
  });
</script>


✅ This prevents double-advance even if a second click handler is still present.

A4) Remove duplicate JS listeners (Buyer Edit only)

B3) Buyer Edit Livewire: deletePersonalPhoto (safe)
public function deletePersonalPhoto(): void
{
    $auction = $this->getAuctionModel(); // however buyer edit loads it
    $path = $auction->personal_photo_path ?? null;

    if (!$path) {
        $this->existingPersonalPhotoPath = null;
        return;
    }

    if (\Illuminate\Support\Facades\Storage::exists($path)) {
        \Illuminate\Support\Facades\Storage::delete($path);
    }

    $auction->personal_photo_path = null;
    $auction->save();

    $this->existingPersonalPhotoPath = null;
    $this->personal_photo = null;

    $this->dispatchBrowserEvent('notify', ['type' => 'success', 'message' => 'Photo deleted.']);
}

Draft-safe safeguard

Only set personal_photo_path = null when user clicks delete.
Do not overwrite it during normal saves unless a new upload replaces it.
