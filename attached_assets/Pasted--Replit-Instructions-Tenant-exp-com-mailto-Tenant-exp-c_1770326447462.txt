## Step 1 — Define the canonical label text (Tenant-only)

Create a Tenant-only config/constant so all pages pull from the same strings.

### Option A (recommended): Config file

`config/tenant_labels.php`

```php

  'lease_option' => [
    'comp_create' => 'Compensation for Creating the Lease-Option Agreement:',
    'comp_exercised' => 'Compensation if Purchase Option is Exercised:',
  ],
```

**Safeguard:** This is additive and won’t affect other roles.
